# VIIVET: Maison Tropicale - Brand Kit

## 1. Brand Identity & Aesthetic
**Brand Name:** VIIVET
**Collection Name:** Maison Tropicale
**Aesthetic:** Quiet luxury, tropical architectural minimalism, high-end editorial. The brand focuses on seamless cinematic experiences, utilizing sleek glassmorphism, dynamic micro-animations, and asymmetrical "scrollytelling" layouts. 

## 2. Color Palette
The VIIVET color system is designed to evoke a rich, warm, and sophisticated tropical resort feel.

*   **Primary Accent (Amber Gold):** `#C8A84B` 
    *   *Usage:* Interactive cursors, borders, animated UI elements, secondary "Maison Tropicale" wordmarks.
*   **Primary Dark (Deep Brown):** `#1A0E05`
    *   *Usage:* Core dark aesthetic backgrounds, primary button backgrounds, luxury dark text contrast.
*   **Primary Light (Ivory):** `#FAF7F0`
    *   *Usage:* Canvas backgrounds, primary light text, clean negative space.
*   **Secondary Light (Cream):** `#F5EDD8` 
    *   *Usage:* Subtle background variations, faint accent text, scroll indicators.

 *(Note: Individual products use dynamic theme colors extracted from their hero images that override these defaults for bespoke showroom experiences).*

## 3. Typography Hierarchy
VIIVET relies on high-contrast dual typography, stripping away basic web fonts in favor of curated Google Fonts that balance classic elegance with modern minimalism.

*   **Header / Display Font:** `Cormorant Garamond` (Serif)
    *   *Usage:* `H1`, `H2`, and the main "VIVET" logotype overlay. 
    *   *Styling:* Rendered with slight letter spacing and a light font-weight (300) for a commanding, classic boutique fashion house aesthetic.
*   **Body / UI Font:** `Outfit` (Sans-Serif)
    *   *Usage:* "Buy Now" CTA buttons, specifications, numbers, and micro-copy. 
    *   *Styling:* Strictly formatted using very wide letter-spacing (`tracking-[0.25em]` to `0.5em`) and uppercase transformations to mimic the minimalism of modern luxury labels.

## 4. UI Patterns & Key Elements
*   **Ghost Buttons:** Transparent rectangles with sharp `1px` faint borders that fill with solid contrasting colors upon hover.
*   **Glassmorphism Plates:** Softly blurred backdrop filters (`backdrop-blur-24px`) layered underneath high-contrast text overlaying cinematic imagery.
*   **The Custom Cursor:** Default browser cursors are disabled and replaced by a trailing elegant ring-and-dot system that morphs in size during interactive hover states.
